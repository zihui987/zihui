# 画布工作流规则

## 一、定位

`D:\Tools\habi-studio` 是哈比星球画布生产工作台。

它负责：

- 读取项目权威资料
- 组织画布节点执行
- 生成中间产物
- 输出自检回执

它不负责：

- 定义事实
- 定义规则
- 直接放行

## 二、权威边界

- `E:\哈比星球\事实层\` 是唯一事实源，只读
- `E:\哈比星球\规则层\` 是唯一规则源，只读
- `E:\哈比星球\交付层\` 是唯一产物回写区
- 画布工具不得直接改写事实层和规则层

## 三、标准执行链

固定执行链：

`夏夏派单 -> habi-studio 节点执行 / 夏仁执行 -> 自检回执 -> 夏审核核 -> 夏夏归档`

## 四、节点契约

每个画布节点至少要有：

- `task_id`
- `node_id`
- `required_inputs`
- `expected_outputs`
- `acceptance_criteria`
- `status`
- `summary`
- `files_changed`
- `blockers`

## 五、产物落盘

画布相关产物统一回写到：

- `E:\哈比星球\交付层\画布产物\`
- `E:\哈比星球\交付层\派单\`
- `E:\哈比星球\交付层\审核\`
- `E:\哈比星球\交付层\归档\`

不得只停留在画布数据库里，不得只停留在聊天里。

## 六、自动绑定规则

夏夏可按 `FILE_INDEX.md` 自动绑定事实依据和规则依据，但只限：

- 依据单一
- 路径明确
- 不涉及事实层或规则层修改

超出以上条件，必须退回补件。

## 七、失败判定

以下任一情况，视为画布链失败：

- 节点未写明输入和输出
- 有产物但无自检回执
- 有回执但无依据路径
- 画布产物未回写交付层
- 事实层或规则层被工具直接改写

## 八、小说到视频提示词无限画布链

当前画布主链固定为：

`小说/剧本输入 -> 15秒单元分镜包 -> GPT Image 2故事板提示词 -> 故事板图 -> 视频提示词 -> 视频片段 -> 剪辑/归档`

各节点职责如下：

| 节点 | 输入 | 输出 | 不允许做的事 |
|------|------|------|--------------|
| 小说/剧本输入 | 小说正稿、EP1-EP6小说修订版、短漫剧剧本定稿版 | 本集事件、情绪曲线、人物关系状态 | 直接跳到出图提示词 |
| 15秒单元分镜包 | 本集事件、短漫剧剧本 | 单元编号、镜号、时长、画面、对白/内心、音效、钩子 | 把小说原文直接当分镜 |
| GPT Image 2故事板提示词 | 分镜包、角色锚点、项链阶段、场景锚点 | 2x2或2x3故事板提示词 | 改写角色/世界观事实 |
| 故事板图 | GPT Image 2提示词、参考图 | 可审核故事板图 | 跳过角色/项链/画风自检 |
| 视频提示词 | 已审故事板图、分镜包 | 15秒视频动态提示词 | 把故事板标注文字生成进成片 |
| 视频片段 | 视频提示词、故事板图 | 15秒视频片段 | 无回执直接进入发布 |

## 九、画布节点验收闸门

每个 15 秒单元进入下一节点前，必须满足：

- 事实依据已绑定：世界观、角色设定、小说/剧本输入至少各一项。
- 角色锚点已绑定：出场角色必须列出物种、毛色、眼睛、服装、关键道具。
- 项链阶段已绑定：EP1治疗前、EP1治疗后至EP23、EP24 三阶段必须写清。
- 前二十集不认家已检查：台词、服装、动作、道具均不得表现小乐完全认家。
- 正片画风已绑定：皮克斯3D动画电影质感用于角色设定图、成片关键帧、即梦输入关键帧；手绘故事板风格只用于故事板页。
- 输出已回写交付层：分镜包、故事板提示词、视频提示词、审核结论必须可追溯。

## 十、自检回执标准格式

每个15秒单元完成节点执行后，必须输出以下格式的自检回执。PASS/FAIL 逐项标记，不可缺项。

```text
## 自检回执

节点：[15秒单元分镜包 / GPT Image 2故事板提示词 / 故事板图 / 视频提示词 / 视频片段]
EP/单元：[EP编号]-[单元编号]
执行人：[夏仁 / 画布节点]
时间：[YYYY-MM-DD HH:MM]

| 检查项 | PASS/FAIL | 依据路径 | 备注 |
|--------|:---------:|---------|------|
| 事实依据已绑定 | PASS/FAIL | `事实层/...` | |
| 角色锚点已绑定 | PASS/FAIL | `规则层/...` 或 `知识库/03_IP与角色/角色视觉锚点.md` | |
| 项链阶段已绑定 | PASS/FAIL | EP1治疗前 / EP1治疗后至EP23 / EP24 | |
| 前二十集不认家已检查 | PASS/FAIL | 台词/服装/动作/道具无认家表现 | |
| 正片画风已绑定 | PASS/FAIL | 皮克斯3D（正片）/ 手绘分镜（故事板页） | |
| 产物已回写交付层 | PASS/FAIL | `交付层/画布产物/...` | 实际路径 |
| 阻塞项 | — | — | 无 / 列出阻塞原因 |

结论：PASS / FAIL（任一 FAIL 不得进入下一节点）
```

## 十一、审核结论独立落盘

1. 每笔正式生产单的审核结论必须独立落盘为 `交付层\审核\TASK-YYYYMMDD-NNN_审核结论.md`
2. 不接受仅在 task log 中存在审核结论
3. 归档前核对时，审核结论文件必须按文件名逐项核到
